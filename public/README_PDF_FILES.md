# PDF файлы для сайта

В этой папке должны находиться следующие PDF файлы:

## Необходимые файлы:

1. **politika_obrabotki_i_zashiti_dannih.pdf**
   - Используется для ссылки "обработкой персональных данных" в форме контактов
   - URL: `/politika_obrabotki_i_zashiti_dannih.pdf`

2. **oferta_2.pdf**
   - Используется для ссылки "Условия использования" в футере
   - URL: `/oferta_2.pdf`

## Инструкция:

1. Поместите реальные PDF файлы в папку `public/`
2. Убедитесь, что имена файлов точно совпадают с указанными выше
3. После добавления файлов пересоберите проект: `npm run build`
4. Файлы будут доступны по соответствующим URL после деплоя

## Важно:

- Файлы из папки `public/` автоматически копируются в `dist/` при сборке
- Убедитесь, что файлы имеют правильную кодировку и корректно открываются в браузере
- Рекомендуемый размер файлов: до 5MB для быстрой загрузки
