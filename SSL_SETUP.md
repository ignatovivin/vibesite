# Настройка SSL сертификата для takto.space

## Проблема: "Незащищенный HTTPS"

Если браузер показывает предупреждение о незащищенном соединении, это означает, что SSL сертификат не установлен или недействителен.

## Решение: Установка SSL сертификата

### Шаг 1: Войдите в панель управления REG.Ru

1. Откройте https://www.reg.ru/user/
2. Войдите в свой аккаунт

### Шаг 2: Найдите раздел SSL-сертификаты

1. В панели управления найдите раздел **"Хостинг"** → **"Управление"**
2. Выберите домен `takto.space`
3. Найдите раздел **"SSL-сертификаты"** или **"Защищенное соединение (SSL)"**

### Шаг 3: Установите Let's Encrypt сертификат

1. Нажмите **"Установить SSL-сертификат"** или **"Активировать SSL"**
2. Выберите **"Let's Encrypt"** (бесплатный сертификат)
3. Выберите домен `takto.space` и поддомен `www.takto.space` (если нужно)
4. Нажмите **"Установить"** или **"Активировать"**

### Шаг 4: Включите принудительное перенаправление на HTTPS

1. В настройках сайта найдите опцию **"Перенаправлять HTTP-запросы в HTTPS"**
2. Включите эту опцию
3. Сохраните изменения

**Примечание:** Редирект на HTTPS также настроен в файле `.htaccess`, который уже загружен на сервер.

### Шаг 5: Подождите активации

- Установка сертификата обычно занимает **5-15 минут**
- После установки сайт будет доступен по HTTPS с зеленым замочком

## Проверка SSL сертификата

### Через браузер:

1. Откройте https://takto.space
2. Нажмите на замочек в адресной строке
3. Проверьте информацию о сертификате:
   - Должен быть указан **"Let's Encrypt"** или другой доверенный центр
   - Должна быть дата окончания действия (обычно 90 дней)
   - Должен быть статус **"Действителен"**

### Через онлайн-сервисы:

- https://www.ssllabs.com/ssltest/analyze.html?d=takto.space
- https://www.sslshopper.com/ssl-checker.html#hostname=takto.space

## Возможные проблемы

### Проблема 1: Сертификат не устанавливается

**Решение:**
- Проверьте, что DNS записи правильно настроены
- Убедитесь, что домен указывает на правильный IP (`31.31.196.49`)
- Подождите 15-30 минут после настройки DNS

### Проблема 2: Смешанный контент (Mixed Content)

Если на HTTPS странице загружаются HTTP ресурсы, браузер покажет предупреждение.

**Решение:**
- Убедитесь, что все ссылки в коде используют `https://` или относительные пути
- Проверьте консоль браузера (F12) на наличие ошибок "Mixed Content"

### Проблема 3: Сертификат истек

Let's Encrypt сертификаты действительны 90 дней и автоматически обновляются.

**Решение:**
- REG.Ru обычно обновляет сертификаты автоматически
- Если сертификат истек, обратитесь в поддержку REG.Ru

## После установки SSL

После успешной установки SSL сертификата:

1. ✅ Сайт будет доступен по HTTPS
2. ✅ Браузер покажет зеленый замочек
3. ✅ Все HTTP запросы будут перенаправляться на HTTPS
4. ✅ Сайт будет безопасным для пользователей
